# Igreja Batista Anseio - Site

Site responsivo e moderno da Igreja Batista Anseio desenvolvido em HTML, CSS e JavaScript.

## 🚀 Características

- ✅ Design moderno e responsivo
- ✅ Menu mobile interativo
- ✅ Seção sobre a igreja
- ✅ Informações de cultos
- ✅ Notícias e destaques
- ✅ Informações de contato
- ✅ Animações suaves
- ✅ Otimizado para todos os dispositivos

## 📁 Estrutura de pastas

```
Igreja Anseio/
├── index.html              # Página principal
├── server.js              # Servidor Node.js
├── package.json           # Dependências
├── README.md             # Documentação
├── assets/
│   ├── css/
│   │   ├── reset.css     # Reset CSS
│   │   └── style.css     # Estilos principais
│   ├── js/
│   │   └── script.js     # JavaScript do cliente
│   └── images/           # Imagens do site
└── node_modules/         # Dependências instaladas
```

## 🔧 Instalação

### 1. Instalar dependências

```bash
npm install
```

Isso instalará:
- **Express** - Framework web
- **CORS** - Permitir requisições de origens diferentes

### 2. Iniciar o servidor

```bash
npm start
```

Você verá:
```
╔═══════════════════════════════════════╗
║  Igreja Batista Anseio - Site         ║
║  Rodando em: http://localhost:3000    ║
╚═══════════════════════════════════════╝
```

### 3. Abrir o site

Abra `http://localhost:3000` no navegador.

2. Execute `npm install` no servidor
